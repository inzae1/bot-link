# bot-link

카카오톡 공유용 정적 페이지입니다.

## 사용 주소

기본 주소:

```text
https://inzae1.github.io/bot-link/
```

추천 결과 전달:

```text
https://inzae1.github.io/bot-link/?title=인재님을%20위한%20추천&body=추천%20내용
```

`title`과 `body`는 반드시 URL 인코딩해서 전달합니다.

## 카카오 개발자 설정

카카오 디벨로퍼스 앱의 **플랫폼 > Web > 사이트 도메인**에 아래 주소를 등록합니다.

```text
https://inzae1.github.io
```

GitHub 저장소의 **Settings > Pages**에서 배포 원본을 `main` 브랜치의 `/ (root)`로 설정합니다.
